# لوحة تحكم مستقلة - الغلابه

لوحة تحكم إدارية مستقلة لموقع الغلابه الإخباري، يمكن تشغيلها بشكل منفصل عن المشروع الرئيسي.

## المميزات

- 🎯 لوحة تحكم كاملة مع إحصائيات الموقع
- 📊 رسوم بيانية تفاعلية للزوار
- 📝 إدارة المقالات
- 👥 نظام مستخدمين مع صلاحيات
- 🌐 دعم ثنائي اللغة (العربية والإنجليزية)
- 🔐 نظام مصادقة آمن
- 📱 واجهة متجاوبة تعمل على جميع الأجهزة

## المتطلبات

- Node.js 18 أو أحدث
- npm أو yarn

## التثبيت والتشغيل

1. انتقل إلى مجلد المشروع:
```bash
cd standalone-admin
```

2. قم بتثبيت الاعتماديات:
```bash
npm install
```

3. قم بتشغيل خادم التطوير:
```bash
npm run dev
```

4. افتح المتصفح على العنوان:
```
http://localhost:3001
```

## هيكل المشروع

```
standalone-admin/
├── src/
│   ├── components/     # المكونات القابلة لإعادة الاستخدام

│   ├── data/          # البيانات الوهمية
│   ├── layouts/       # تخطيطات الصفحات

│   └── main.tsx       # نقطة الدخول

```
